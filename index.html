Grid Sketchpad

Grid Sketchpad Instructions:

  1. Drag your cursor around the grid to start drawing
  2. Click "Clear" to erase the grid
  3. Change the size of the grid by entering a number from 1 to 100. The default size is 20x20
  4. Click "Greyscale" to draw in greyscale
  5. Click "Random" to draw in random colors
